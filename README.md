# Smart_toast


## Gradle 


Step 1. Add the JitPack repository in your root build.gradle at the end of repositories:
    
    
    
    
    allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}



Step 2. Add the dependency

	        implementation 'com.github.developer5167:Smart_toast:1.0.0'
